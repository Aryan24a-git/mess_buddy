# Mess Buddy 🚀

Mess Buddy is a high-performance, offline-first expense tracking and management engine designed specifically for hostel and mess environments. Built using Flutter, it utilizes Clean Architecture and Riverpod for state management.

## 🌟 Features
- **Offline-First:** Fully powered by SQLite, meaning zero latency and zero dependency on the internet.
- **Glassmorphic UI Engine:** Built entirely from scratch adhering to a custom dark-mode stitch design system with native 8pt clipping.
- **Goals & Alerts:** Eagerly tracks and projects burn rates, alerting you via native OS notifications when nearing your limits.
- **Roommate Splitter:** Splitting logic calculator for instant roommate settlements.
- **PDF Export Engine:** Generate & share pristine analytics reports.
- **Built-in Monetization:** Deep integration with AdMob and a mock "Pro" tier upgrader.

## 🛠️ Tech Stack
*   **Framework:** Flutter (Dark Mode Custom UI)
*   **State Management:** Riverpod
*   **Database:** SQLite (`sqflite`)
*   **Routing:** GoRouter
*   **Others:** `google_mobile_ads`, `flutter_local_notifications`, `pdf`, `printing`
